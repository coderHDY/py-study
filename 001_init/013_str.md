# Python 3 字符串（str）完整语法参考

## 一、字符串创建

| 写法 | 说明 | 示例 |
|------|------|------|
| `'text'` / `"text"` | 单/双引号 | `s = 'hello'` |
| `'''text'''` / `"""text"""` | 多行字符串 | `s = """line1\nline2"""` |
| `r'text'` | 原始字符串，不转义 | `r'\n'` → 两个字符 `\n` |
| `f'text'` | f-string 格式化 | `f'hi {name}'` |
| `b'text'` | 字节串（bytes） | `b'hello'` |
| `str(obj)` | 将对象转为字符串 | `str(123)` → `'123'` |

---

## 二、切片（Slice）

语法：`s[start:stop:step]`

| 操作 | 说明 | 示例（s = 'hello'） |
|------|------|------|
| `s[i]` | 取第 i 个字符（负数从尾） | `s[1]` → `'e'`，`s[-1]` → `'o'` |
| `s[start:stop]` | 取 [start, stop) 子串 | `s[1:3]` → `'el'` |
| `s[start:]` | 从 start 到末尾 | `s[2:]` → `'llo'` |
| `s[:stop]` | 从开头到 stop | `s[:3]` → `'hel'` |
| `s[::step]` | 按步长取字符 | `s[::2]` → `'hlo'` |
| `s[::-1]` | 字符串反转 | `s[::-1]` → `'olleh'` |
| `s[1:4:2]` | start/stop/step 全用 | `s[1:4:2]` → `'el'` |

---

## 三、运算符

| 操作 | 说明 | 示例 |
|------|------|------|
| `s1 + s2` | 拼接 | `'hi' + ' you'` → `'hi you'` |
| `s * n` | 重复 | `'ab' * 3` → `'ababab'` |
| `c in s` | 子串判断 | `'el' in 'hello'` → `True` |
| `c not in s` | 不包含判断 | `'x' not in 'hello'` → `True` |
| `len(s)` | 长度 | `len('hello')` → `5` |

---

## 四、查找 / 判断方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `s.find(sub[, start, end])` | 返回首次出现位置，找不到返回 -1 | `'hello'.find('ll')` → `2` |
| `s.rfind(sub)` | 从右侧查找，返回最后位置 | `'hello'.rfind('l')` → `3` |
| `s.index(sub)` | 同 find，找不到抛 ValueError | `'hello'.index('e')` → `1` |
| `s.rindex(sub)` | 从右查找，找不到抛异常 | `'hello'.rindex('l')` → `3` |
| `s.count(sub)` | 统计子串出现次数 | `'hello'.count('l')` → `2` |
| `s.startswith(prefix)` | 是否以前缀开头 | `'hello'.startswith('he')` → `True` |
| `s.endswith(suffix)` | 是否以后缀结尾 | `'hello'.endswith('lo')` → `True` |

---

## 五、大小写方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `s.upper()` | 全大写 | `'hello'.upper()` → `'HELLO'` |
| `s.lower()` | 全小写 | `'HELLO'.lower()` → `'hello'` |
| `s.title()` | 每个单词首字母大写 | `'hello world'.title()` → `'Hello World'` |
| `s.capitalize()` | 首字母大写，其余小写 | `'hELLO'.capitalize()` → `'Hello'` |
| `s.swapcase()` | 大小写互换 | `'Hello'.swapcase()` → `'hELLO'` |
| `s.casefold()` | 更激进的小写（用于比较） | `'ß'.casefold()` → `'ss'` |

---

## 六、去除空白 / 填充方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `s.strip([chars])` | 去除两端空白或指定字符 | `'  hi  '.strip()` → `'hi'` |
| `s.lstrip([chars])` | 去除左侧空白或指定字符 | `'  hi'.lstrip()` → `'hi'` |
| `s.rstrip([chars])` | 去除右侧空白或指定字符 | `'hi  '.rstrip()` → `'hi'` |
| `s.center(width[, fill])` | 居中，填充字符 | `'hi'.center(6, '-')` → `'--hi--'` |
| `s.ljust(width[, fill])` | 左对齐，右侧填充 | `'hi'.ljust(5, '.')` → `'hi...'` |
| `s.rjust(width[, fill])` | 右对齐，左侧填充 | `'hi'.rjust(5, '.')` → `'...hi'` |
| `s.zfill(width)` | 左侧补零 | `'42'.zfill(5)` → `'00042'` |

---

## 七、替换 / 分割 / 拼接方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `s.replace(old, new[, count])` | 替换子串 | `'aabbaa'.replace('aa', 'x')` → `'xbbx'` |
| `s.split([sep, maxsplit])` | 按分隔符拆分为列表 | `'a,b,c'.split(',')` → `['a','b','c']` |
| `s.rsplit([sep, maxsplit])` | 从右侧拆分 | `'a,b,c'.rsplit(',', 1)` → `['a,b','c']` |
| `s.splitlines([keepends])` | 按行拆分 | `'a\nb'.splitlines()` → `['a','b']` |
| `sep.join(iterable)` | 用 sep 连接可迭代对象 | `','.join(['a','b','c'])` → `'a,b,c'` |
| `s.partition(sep)` | 按首次 sep 分为三元组 | `'a:b:c'.partition(':')` → `('a',':','b:c')` |
| `s.rpartition(sep)` | 从右侧按 sep 分三元组 | `'a:b:c'.rpartition(':')` → `('a:b',':','c')` |

## 八、判断类型方法（返回 bool）

| 方法 | 作用 | 示例 |
|------|------|------|
| `s.isalpha()` | 全为字母 | `'abc'.isalpha()` → `True` |
| `s.isdigit()` | 全为数字（0-9） | `'123'.isdigit()` → `True` |
| `s.isnumeric()` | 全为数字字符（含汉字数字） | `'①'.isnumeric()` → `True` |
| `s.isdecimal()` | 全为十进制数字 | `'123'.isdecimal()` → `True` |
| `s.isalnum()` | 全为字母或数字 | `'abc123'.isalnum()` → `True` |
| `s.isspace()` | 全为空白字符 | `'   '.isspace()` → `True` |
| `s.isupper()` | 全为大写字母 | `'ABC'.isupper()` → `True` |
| `s.islower()` | 全为小写字母 | `'abc'.islower()` → `True` |
| `s.istitle()` | 是否标题格式 | `'Hello World'.istitle()` → `True` |
| `s.isidentifier()` | 是否合法标识符 | `'myVar'.isidentifier()` → `True` |
| `s.isprintable()` | 是否全部可打印 | `'hello'.isprintable()` → `True` |
| `s.isascii()` | 是否全为 ASCII 字符 | `'hello'.isascii()` → `True` |

---

## 九、格式化方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `s.format(*args, **kwargs)` | 格式化字符串 | `'{} + {} = {}'.format(1, 2, 3)` → `'1 + 2 = 3'` |
| `s.format_map(mapping)` | 用字典格式化 | `'{name}'.format_map({'name': 'Tom'})` → `'Tom'` |
| `f'{expr}'` | f-string（推荐） | `f'{1+1}'` → `'2'` |
| `f'{val:.2f}'` | f-string 格式规范 | `f'{3.14159:.2f}'` → `'3.14'` |
| `f'{val:>10}'` | 右对齐，宽度 10 | `f'{"hi":>10}'` → `'        hi'` |
| `f'{val:0>5}'` | 左侧补 0 | `f'{42:0>5}'` → `'00042'` |
| `'%s %d' % (s, n)` | 旧式格式化 | `'%s=%d' % ('x', 5)` → `'x=5'` |

---

## 十、编码 / 解码

| 方法 | 作用 | 示例 |
|------|------|------|
| `s.encode([encoding])` | 字符串→字节串 | `'你好'.encode('utf-8')` → `b'\xe4\xbd\xa0\xe5\xa5\xbd'` |
| `b.decode([encoding])` | 字节串→字符串 | `b'hello'.decode('utf-8')` → `'hello'` |

---

## 十一、其他常用内置函数（配合字符串）

| 函数 | 作用 | 示例 |
|------|------|------|
| `len(s)` | 字符串长度 | `len('hello')` → `5` |
| `ord(c)` | 字符→Unicode 码点 | `ord('A')` → `65` |
| `chr(n)` | Unicode 码点→字符 | `chr(65)` → `'A'` |
| `repr(s)` | 返回可打印的字符串表示 | `repr('hi\n')` → `"'hi\\n'"` |
| `input([prompt])` | 从控制台读取字符串 | `s = input('请输入: ')` |
| `print(*args)` | 输出字符串 | `print('hello', 'world')` |
| `sorted(s)` | 字符排序 | `sorted('bac')` → `['a','b','c']` |
| `reversed(s)` | 返回反向迭代器 | `''.join(reversed('abc'))` → `'cba'` |
| `enumerate(s)` | 返回索引+字符 | `list(enumerate('ab'))` → `[(0,'a'),(1,'b')]` |
